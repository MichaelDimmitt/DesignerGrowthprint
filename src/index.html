🔧 How Template Loading Works

Inline templates: Content lives in <template> tags in this HTML file. When activated, the content is cloned via template.content.cloneNode(true) and inserted into the DOM.

External pages: HTML is fetched, parsed into a temporary <template> element, then cloned and inserted. This ensures scripts execute properly when content enters the live DOM.
